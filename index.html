Project

Automatic Trash Based on Arduino

The Arduino-based automatic trash can uses an Arduino Mega 2560, speaker, mini DF player, servo and ultrasonic sensor. When at a distance of 10 cm the sensor will detect and the trash can will automatically open along with the sound of the speaker, when the object is out of range (10cm) the trash can will close automatically along with the sound of the speaker.

Bluetooth Control Car Based on Arduino

Bluetooth remote control car using Arduino Mega 2560, HC 05 Bluetooth model, L2896 motor driver, ultrasonic sensor. This car has 2 modes, the first mode uses a distance sensor, with this mode the car will automatically move forward and avoid the obstacle. the second is Bluetooth mode which can be activated and controlled via the available application to go forward, backward and turn.

Automatic Plant Watering System Based on Arduino

Automatic plant waterer using an Arduino Mega 2560, soil moisture sensor, relay, and electric pump. The sensor is plugged into the ground to measure the humidity of the soil. When the humidity of a plant is low, the relay will turn on and the pump will release water to water the plant. When the humidity is right, the relay turns off and the pump also turns off.

Learning Media Application “SI HUJAN“

The learning media application "SI HUJAN" is .apk based which can be installed on Android. Created using the smart apps creator, there are 3 menus, materi menu containing explanations about the occurrence of rain, a quiz menu containing interactive and interesting quizzes about the occurrence of rain, and a guide menu for using the "SI HUJAN" application.
